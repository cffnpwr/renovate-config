# cffnpwr/renovate-config

[![GitHub License](https://img.shields.io/github/license/cffnpwr/eslint-config?style=flat)](./LICENSE)

A shared Renovate configuration preset for cffnpwr.

[日本語のREADMEはこちら](./README-ja.md)

## How to Use

Set up `renovate.json` in your repository as follows.

```json
{
  "$schema": "https://docs.renovatebot.com/renovate-schema.json",
  "extends": [
    "github>cffnpwr/renovate-config"
  ]
}
```
